# Contributing to PhishingShield

Welcome! We appreciate your interest in contributing to PhishingShield.

## How to Contribute

### 1. Reporting Bugs
* Check issues to see if it has already been reported.
* Open a new issue using our Bug Report template.

### 2. Suggesting Enhancements
* Open an issue describing the feature proposal.

### 3. Pull Requests
* Fork the repository.
* Create a feature branch: `git checkout -b feature/my-cool-feature`.
* Ensure tests pass: `pytest`.
* Open a Pull Request referencing the original issue.
