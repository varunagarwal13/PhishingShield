# Security Policy

## Supported Versions

We support security patches for the latest release:

| Version | Supported |
| :--- | :--- |
| v3.x |  Yes |
| < v3.x | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities publicly in issues. Send vulnerabilities via email to varun.agarwal@example.com (replace with proper address). We will respond within 48 hours to coordinate a security advisory.
